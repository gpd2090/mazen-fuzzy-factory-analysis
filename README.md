# Maven Fuzzy Factory Analysis

## Project Overview
This project analyzes marketing performance, revenue trends, product profitability, and refund impact for a simulated e-commerce company.

## Tools Used
- PostgreSQL
- Power BI
- Excel

## Key Analysis Areas
- Channel Performance
- Conversion Rate Analysis
- Revenue & Profitability
- Product Performance
- Time-Based Trends
- Refund Analysis

## Key Insights
- Brand campaigns deliver highest ROI
- Socialbook pilot underperforms
- Revenue growth driven by traffic increase
- Certain products have higher profit margins
- Refunds impact net revenue trends
